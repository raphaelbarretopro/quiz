Importador de Aulas

Ferramenta gráfica para colar o JSON de uma nova aula, validar a estrutura esperada e inserir essa aula no course-data.json no formato atual do projeto. Se o navegador suportar File System Access API, também é possível salvar direto no arquivo local.

1. Base do Curso

Carregue o course-data atual do projeto ou selecione um arquivo local.

A ferramenta mantém o formato com identação de 2 espaços.

2. Nova Aula

Cole aqui o JSON da aula nova no formato antigo (lesson_info + questions) ou o objeto completo da aula.

Status

Aguardando carregamento do course-data e colagem da nova aula.

Resumo da Aula

  • Nenhuma aula validada ainda.

Resultado

Depois da mescla, use uma das opções abaixo para persistir o arquivo atualizado.

O botão "Salvar no arquivo local" funciona apenas em navegadores com suporte à File System Access API, como Chromium/Edge recentes.

Regras de Validação

  • lesson_info.id, lesson_info.title e topics são obrigatórios.
  • topics precisa ter itens com id, name, desc, color, img_url e sokoban.
  • questions precisa existir e não pode ser vazio.
  • O slug pode ser informado manualmente ou derivado de lesson_info.id.
  • Os tipos suportados são: multiple, boolean, combo, multi e drag.
