Contexto: 
Atua como um Especialista em Design Instrucional e Desenvolvedor Full-stack. Preciso que transformes o conteúdo da aula anexa (PDF/Texto) em um novo objeto JSON para ser inserido no meu arquivo de curso.

Diretrizes Estruturais:
Estrutura de Tópicos: Cria 4 tópicos (ACID, AGEN, PROT, NORM) baseados no conteúdo anexo. Cada um deve ter um id, name, desc (curto), color (hexadecimal vibrante) e uma img_url do Unsplash relacionada.

Sokoban: Cria um array sokoban com 4 letras que formem uma palavra-chave central desta aula (ex: para Redes seria ["H", "U", "B", "S"]).
Questões (Obrigatório): Gera um total de 50 questões distribuídas obrigatoriamente pelos seguintes tipos:
multiple: Escolha única.
boolean: Verdadeiro ou Falso.
combo: Frase com lacuna [COMBO] para preencher.
multi: Múltipla seleção (várias corretas).
drag: Arrastar e soltar (Relacionar items com zones).
Personalização e Feedback:
Todas as perguntas (onde aplicável) devem usar a tag [NAME] para se referir ao aluno.
Cada questão deve ter um campo tip com uma explicação pedagógica curta.
A última questão de cada tópico deve ter o campo trans indicando o ID do próximo tópico.
Formato de Saída:
Fornece apenas o código JSON do novo objeto que será inserido no array lessons, seguindo exatamente este padrão:
JSON

{
  "slug": "aula-X",
  "lesson_info": { ... },
  "questions": [ ... ]

}
